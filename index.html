Hi I'm Paul and I work as a UI/UX Developer for 9 years. Since the beginning of my journey, I've been developing Responsive Web Applications, E-commerce site, Web animations, Google Rich media ads and Mobile/Desktop application. That doesn't stop my journey as a developer, as I also create Web design, User Research, Information Architecture, Wireframes and Prototypes. I collaborate with clients and talented people to produce quality and amazing products. I enjoy every aspect of what I do and hopeful to create something that would make people happy.
Featured web applications that I've worked on are Samsung, Intel, L'Oreal, Samsonite, Mentos, Globe, Del Monte, Ayala, Maybelline, Lipton, Watsons, Dove, Garnier, Marlboro and Medicol. Feel free to view all my works.

FRONTEND DEVELOPMENT

UI Development using HTML5, CSS3, Bootstrap, Sass, jQuery, AngularJS, BackboneJS and DNN (DotNetNuke) CMS software development.

UX & DIGITAL DESIGN

Creating Responsive Web Design and User Interface for different platforms. User Research, Information Architecture, Prototypes and Wireframes

ANIMATION

Flash Animation for web apps and ads. HTML5 Animation Using CreateJS. Creating Spritesheets for HTML5 canvas. CSS3 Animation.

MOBILE & DESKTOP APPLICATION

Creating mobile application using Adobe Air. Using Starling feathers library fast GPU powered graphics and an extensible framework architeture.
